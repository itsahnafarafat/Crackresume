# **App Name**: ResuAI

## Core Features:

- AI Content Assist: AI-Guided Content Suggestions: The tool analyzes job descriptions and provides tailored content suggestions for experience, skills, and summary sections.
- Keyword Optimizer: Real-time Keyword Optimization: The tool analyzes the resume content, then analyzes the provided job description, and provides real-time feedback on keyword usage to improve relevance.
- ATS Check: ATS Compatibility Checker: Analyzes the resume to ensure it is compatible with Applicant Tracking Systems, highlighting potential issues.
- Template Library: Customizable Templates: Offers a variety of professional templates that can be easily customized to fit different job types and personal preferences.
- Resume Editor: User-Friendly Editor: An intuitive drag-and-drop interface that allows users to easily edit and rearrange sections of their resume.

## Style Guidelines:

- Primary color: Professional blue (#29ABE2) to convey trust and competence.
- Background color: Clean, off-white background (#F9F9F9) for readability.
- Accent color: A brighter shade of blue (#037DBE) to highlight key action items and sections.
- Body and headline font: 'Inter', a sans-serif font known for its readability and clean, modern appearance. Suitable for both headlines and body text.
- Simple, professional icons to represent different sections and features (e.g., experience, skills).
- Clean and structured layout with clear visual hierarchy for easy scanning and readability.
